JavaScript

JavaScript é uma linguagem de programação de alto nível e de script que permite interatividade dinâmica nos navegadores. O básico inclui variáveis, estruturas de controle de fluxo e funções.

Principais Comandos JavaScript

                        
                            // Declaração de variáveis
                            let nome = "Usuário";
                            const PI = 3.14;
    
                            // Estruturas de Controle de Fluxo
                            if (condicao) {
                                // código a ser executado se a condição for verdadeira
                            } else {
                                // código a ser executado se a condição for falsa
                            }
    
                            for (let i = 0; i < 5; i++) {
                                // código a ser repetido
                            }
    
                            // Funções
                            function saudacao() {
                                return "Olá, " + nome + "!";
                            }
    
                            // Chamada de Função
                            let mensagem = saudacao();
                            console.log(mensagem);

CSS

CSS, que significa Folhas de Estilo em Cascata, é usado para estilizar e formatar o conteúdo HTML. Ele controla o layout, as cores, as fontes e outros aspectos visuais de uma página web.

Principais Comandos e Dicas de CSS

                        
                            /* Estilizar um elemento */
                            selector {
                                property: value;
                            }
    
                            /* Selecionar por classe */
                            .minha-classe {
                                /* estilos aqui */
                            }
    
                            /* Selecionar por ID */
                            #meu-id {
                                /* estilos aqui */
                            }
    
                            /* Estilizar links */
                            a {
                                text-decoration: none;
                                color: #3498db; /* Azul */
                                transition: color 0.3s ease;
                            }
    
                            a:hover {
                                color: #e74c3c; /* Vermelho quando hover */
                            }
    
                            /* Layout Responsivo */
                            @media screen and (max-width: 600px) {
                                /* Estilos para telas menores */
                            }

Dicas para tornar o site atraente:

  • Escolha uma paleta de cores harmoniosa.
  • Utilize fontes legíveis.
  • Adicione espaçamento adequado para melhor legibilidade.
  • Use imagens e ícones de forma equilibrada.
  • Adote um layout responsivo para diferentes dispositivos.

Atividade Prática e Projeto Simples

A atividade prática consiste na criação de um site simples utilizando os conceitos de HTML, CSS e JavaScript. Os alunos serão desafiados a aplicar os conhecimentos adquiridos em sala de aula para desenvolver uma página web funcional.

O projeto incluirá a estruturação do conteúdo com HTML, a estilização usando CSS para tornar o site visualmente atraente e a adição de interatividade dinâmica com JavaScript. Serão encorajados a explorar diferentes elementos e técnicas para personalizar o projeto de acordo com suas preferências.

A ênfase será na prática dos conceitos fundamentais, proporcionando aos alunos a oportunidade de consolidar seus conhecimentos e desenvolver habilidades práticas essenciais para a área de desenvolvimento web.
